About DriftCoin

What is DriftCoin?

DriftCoin is a BSC EIP-20 Token based which is available for multi-chain transaction and tradeable on DeFi ecosystem.

How 3*3% AirDrop work?

Whenever we reach a possible AirDrop checkpoint and target Hodler count reached, we will equal distribute token between the holders.
In case of target Holder count is not reached, the AirDrop will be burned!
